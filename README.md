ผมเป็นคนประเภทที่เห็นปริศนาหรือปัญหาอะไรแปลกๆ ไม่ได้ ต้องวิ่งเข้าใส่เพื่อหาคำตอบเสมอ ตั้งแต่เด็กผมชอบตั้งคำถามกับตัวเองว่า “นั่นเกิดขึ้นได้ยังไง?” หรือ “ทำไมระบบนี้ถึงทำงานแบบนั้น?” หลายคนอาจจะมองว่าเป็นคำถามที่ไร้สาระ แต่สำหรับผม การชอบหาเหตุผลแบบนี้แหละที่สร้างตัวตนของผมขึ้นมา ตอนแรกผมก็ยังไม่รู้ว่าตัวเองอยากเป็นอะไรในอนาคต จนกระทั่งผมได้มารู้จักกับการเขียนโค้ด (Coding) มันเหมือนผมได้เจอจิ๊กซอว์ชิ้นสุดท้าย เพราะคอมพิวเตอร์คือโลกที่ทุกอย่างขับเคลื่อนด้วยเหตุและผล และเป็นพื้นที่ที่ผมสามารถใช้ทักษะการแก้ปัญหาของผมได้อย่างเต็มที่ที่สุด

พอรู้ตัวว่าชอบอะไร ผมเลยตัดสินใจทุ่มเทเวลาศึกษาเรื่องนี้อย่างจริงจังด้วยตัวเอง เริ่มลุยตั้งแต่ JavaScript, Python, Java ไปจนถึง HTML และ CSS ยิ่งเรียนลึกขึ้น ผมยิ่งมองเห็นว่าโค้ดไม่ใช่แค่ตัวหนังสือที่พิมพ์ลงในจอ แต่โค้ดคือทุกสิ่งรอบตัวเรา มันคือกระบวนการคิดและระบบการทำงานของโลกใบนี้ ทุกอย่างมันถูกจัดการอย่างเป็นระเบียบผ่านหลักการ Input, Process และ Output ซึ่งการที่เราสามารถเปลี่ยนไอเดียในหัวให้กลายเป็นระบบที่ทำงานได้จริง คือเสน่ห์ที่ทำให้ผมหลงรักศาสตร์นี้ และอยากจะเก่งขึ้นในทุกๆ วัน

สาขาวิชาวิทยาการคอมพิวเตอร์ ของสถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง คือที่ที่ตอบโจทย์ชีวิตผมมากที่สุด เพราะผมรู้ว่าที่นี่ขึ้นชื่อเรื่องความเข้มข้น และเป็นศูนย์รวมของคนที่บ้าคลั่งในสิ่งเดียวกัน ผมมั่นใจว่าแพสชันที่ผมมี บวกกับพื้นฐานโค้ดดิ้งที่ผมฝึกฝนมา จะเข้ากันได้ดีกับหลักสูตรของ สจล. และผมพร้อมมากที่จะเข้ามาเก็บเกี่ยวความรู้ พัฒนาทักษะของตัวเองให้ไปสู่ระดับที่สูงขึ้น เพื่อจบไปเป็นนักพัฒนาที่สร้างสรรค์เทคโนโลยีที่ใช้งานได้จริงในอนาคตครับ
